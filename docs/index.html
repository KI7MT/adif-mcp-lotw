ADIF MCP LoTW

ADIF MCP LoTW is an open-source project bringing ADIF 3.1.5 logging standards into the Model Context Protocol (MCP) ecosystem. It enables AI agents and logging applications to access, validate, and transform amateur radio QSO records through a common, safe interface.
